# Smoker-Generator
Give me photo, I help you smoke!

## Prerequisite

- python3
- opencv
- mtcnn
- numpy
- keras
- tensorflow


They could all be installed via pip.

## Usage
run:
> python3 smoker_generator.py path_to_your_image.jpg
